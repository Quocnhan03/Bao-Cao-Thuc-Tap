### Lab 90 – Amazon Macie: Discover Sensitive Data in S3

**Amazon Macie** is a data security service that uses Machine Learning (ML) to automatically discover, classify, and protect sensitive data (PII, credit cards, etc.) stored in Amazon S3.

**Implementation Steps:**
1. Navigate to **Amazon Macie** in the AWS Console (region: Singapore `ap-southeast-1`).
2. Click **"Get started"** on the welcome page.
3. Review the information explaining Macie's Service-Linked Role permissions.
4. Click **"Enable Macie"** to activate the service with a 30-day free trial.
5. After activation, the system navigates to the **Summary Dashboard** showing an overview of **Automated Discovery** and **Data Security**.

**Results:**
* Successfully enabled Amazon Macie with the Dashboard displaying **Total accounts: 1**.
* The system is ready to scan and discover sensitive data across S3 buckets.

### Lab 96 – AWS Secrets Manager: Manage Credentials for RDS & Fargate

**AWS Secrets Manager** is a service that helps protect sensitive information (database credentials, API keys, OAuth tokens) by storing and managing them securely. Applications can retrieve this information via API instead of hard-coding passwords in source code.

**Implementation Steps:**
1. Navigate to **AWS Secrets Manager** in the AWS Console.
2. Click **"Store a new secret"**.
3. Select Secret type: **"Other type of secret"** (with Key/value pairs).
4. Fill in details:
   - `username` = `admin`
   - `password` = `Admin@123456`
5. Click **"Next"** to proceed to secret name configuration.
6. Enter Secret name: `myapp/rds/credentials`.
7. Enter description: *Database credentials for Fargate application - Lab 96*.
8. Click **"Next"** through Configure Rotation (skip).
9. On the Review page, examine the **Sample code** for application integration.
10. Click **"Store"** to save the Secret.

**Results:**
* Successfully created Secret `myapp/rds/credentials` in AWS Secrets Manager.
* The system displayed the message: *"You successfully stored the secret myapp/rds/credentials"*.
* Sample code is provided to integrate with the Fargate application to retrieve DB credentials without hard-coding.
