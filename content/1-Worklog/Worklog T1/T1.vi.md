## 3. Kết quả đạt được
* **Nền tảng AWS:** Nắm vững các nhóm dịch vụ cốt lõi bao gồm Compute (EC2), Storage (S3, EBS), Networking (VPC) và Database (RDS, DynamoDB).
* **Thiết lập môi trường:** Đăng ký và bảo mật thành công tài khoản AWS Free Tier.
* **AWS Management Console:** Thành thạo việc tìm kiếm, điều hướng và quản lý tài nguyên thông qua giao diện web.
* **AWS CLI:** Hoàn tất cấu hình môi trường (Access Key, Secret Key, Region mặc định) và thực hiện thành thạo các lệnh quản lý tài nguyên (trạng thái EC2, Key Pairs, liệt kê Region).
* **Quản trị EC2:** Khởi tạo thành công các instance Amazon Linux, cấu hình Security Group và thiết lập kết nối SSH sử dụng Key Pair.
* **Web Hosting:** Triển khai và kiểm thử thành công Web Server (Apache/Nginx) có thể truy cập qua Public IP.
* **Quản lý lưu trữ:** Thực hiện thành công việc gắn EBS volume vào EC2, tạo S3 bucket và upload file lên hệ thống.

> **Ghi chú:** Đã rèn luyện được khả năng linh hoạt kết hợp giữa AWS Management Console và AWS CLI để quản lý tài nguyên hiệu quả.
