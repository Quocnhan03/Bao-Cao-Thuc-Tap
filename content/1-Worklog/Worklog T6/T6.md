## 3. Detailed Practice Content

### 3.1. AWS Systems Manager (Lab 31)
**Objective:** Manage EC2 instances securely without SSH, automate command execution.
- Access Systems Manager Dashboard.
- Configure Session Manager to control servers directly from the browser.
- Use Run Command to install applications across multiple EC2 instances without logging into each one.

![AWS SSM](/images/WorklogT6/lab31-ssm-dashboard.png)

### 3.2. AWS Key Management Service (Lab 33)
**Objective:** Create custom encryption keys to protect sensitive data on the Cloud.
- Access the KMS (Key Management Service) interface.
- Initialize a Symmetric Customer Managed Key (CMK).
- Configure KMS Key Policy to grant access permissions.
- Apply the created key to automatically encrypt an S3 bucket.

![AWS KMS](/images/WorklogT6/lab33-kms-create.png)

### 3.3. AWS Cost Explorer (Lab 34)
**Objective:** Read spending charts and track budgets automatically.
- Access the Billing and Cost Management interface.
- Use Cost Explorer to filter costs by service (EC2, S3...) and by region.
- Create AWS Budgets to set a  limit and alert via email if exceeded.

![Cost Explorer](/images/WorklogT6/lab34-cost-explorer.png)
